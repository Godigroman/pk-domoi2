# Краткая инструкция: Как забрать NUC домой

Закройте все программы и сохраните файлы.

Выключите компьютер программно: Пуск → кнопка питания → Завершение работы. Ни в коем случае не дергайте шнур питания во время работы!

Дождитесь полного выключения: Убедитесь, что погасли все индикаторы и вентилятор не шумит (обычно 10–20 секунд).

Отключите все кабели: Питание, монитор, клавиатура, мышь, интернет (LAN).

Аккуратно упакуйте: Положите компьютер в сумку-рюкзак, желательно в отделение с мягкой подкладкой, чтобы не поцарапать корпус и не повредить порты.
